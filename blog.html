Global Scope: A variable declared at the top of a program or outside of a function is considered a global scope variable.
Let's see an example of a global scope variable.

// program to print a text
let a = "hello";
function greet () {
console.log(a);
}
greet(); // hello

Block Scope: Block scopes are what you get when you use if statements, for statements, and the like. You can also use them stand-alone with a simple begin-end curly braces {}, not to be confused with empty object literals.

var a = {} // empty object literal

{ var a } // undefined object in a block scope

if (3 == '3') { // block scope for the if statement }

Event Loop: We know,JavaScript is a single Threaded Language and being single Threaded it's work synchronous way. But after being single Threaded it's also works asynchronous way with the help of Browser and Event Loop.

Event Loop Work: Event loop is a mechanism that allows JavaScript to perform non-blocking operations. When an asynchronous task is started, the event loop will start running. Once the task is completed, the event loop will again check for any other tasks that need to be performed.

if it's asynchronous then it's not console. next synchronous data will be console. with the help of Browser API like (DOM, AJax etc) asynchronous data will be work and browser send them callback queue and the event loop check stack is empty or not. if all synchronous work done and stack is empty then with the help of event loop from callback queue to stack data will transfar and then the data will console.

8 Ways to get undefined: We can get undefined from the 8 reasons below:

1) Variable that is not initialized will give undefined.
2) Function with no return will give undefined.
3) Parameter that is not passed will be undefined.
4) If return has nothing on the right side then will return undefined.
5) Property that doesn't exists on an object will give result undefined.
6) Accessing array elements outside of the index range will give result undefined.
7) Deleting an element inside an array will give undefined.
8) Set a value directly to undefined.
